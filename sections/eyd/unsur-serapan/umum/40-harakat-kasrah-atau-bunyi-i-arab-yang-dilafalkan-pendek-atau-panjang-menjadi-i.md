# 40. Harakat kasrah atau bunyi /i/ (Arab) yang dilafalkan pendek atau panjang menjadi i.

**Misalnya:**

|  |  |  |
| --- | --- | --- |
| *i't**i**kāf* | (اِعْتِكَافٌ) | ikt*i*kaf |
| *q**i**yāmah* | (قِيَامَةٌ) | k*i*amat |
| *naṣ**ī**ḥah* | (نَصِيْحَةٌ) | nas*i*hat |
| *ṣaḥ**ī**ḥ* | (صَحِيْحٌ) | sah*i*h |
