# 4. Huruf hamzah (ء Arab) yang dibaca vokal menjadi a, i, atau u.

**Misalnya:**

|  |  |  |
| --- | --- | --- |
| ***a**mr* | (أَمْرٌ) | *a*mar |
| *mas**'a**lah* | (مَسْئَلَةٌ) | mas*a*lah |
| ***i**syārah* | (إِشَارَةٌ) | *i*syarat |
| *nā**'i**b* | (نَائِبٌ) | na*i*b |
| ***u**fuq* | (أُفُقٌ) | *u*fuk |
| ***u**ṣūl* | (أُصُوْلٌ) | *u*sul |
