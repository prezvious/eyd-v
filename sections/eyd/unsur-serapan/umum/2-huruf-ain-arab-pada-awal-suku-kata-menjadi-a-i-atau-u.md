# 2. Huruf 'ain (ﻉ Arab) pada awal suku kata menjadi a, i, atau u.

**Misalnya:**

|  |  |  |
| --- | --- | --- |
| ***'a**jā'ib* | (عَجَائِبُ) | *a*jaib |
| *sa**'ā**dah* | (سَعَادَةٌ) | sa*a*dah |
| ***'i**lm* | (عِلْمٌ) | *i*lmu |
| *qā**'i**dah* | (قَاعِدَةٌ) | ka*i*dah |
| ***'u**zr* | (عُذْرٌ) | *u*zur |
| *ṭā**'ū**n* | (طَاعُوْنٌ) | ta*u*n |
