|  |  |
| --- | --- |

| *'a**ll**āmah* | a*l*amah |
| *ba**ll**et* | ba*l*et |
| *co**mm**i**ss**ion* | ko*m*i*s*i |

Konsonan rangkap dipertahankan jika menimbulkan ketaksaan atau konotasi negatif.

**Misalnya:**

|  |  |  |
| --- | --- | --- |

| *ma**ss*** | ma*ss*a | (bandingkan dengan*masa*<br>) |
| *te**ll**er* | te*ll*er | (bandingkan dengan*teler*<br>) |
